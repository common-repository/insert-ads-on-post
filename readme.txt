Insert Ads On Post allow you to insert ads in post content or in single post and page post, ad position is above and below post content.

== Description ==

Insert Ads On Post allow you to insert ads in post content or in single post and page post. This plugin has two forms, the first form is used for an ad with the position is above the post content and under the post title, the second form is used for an ad below the post content.

By default the ads appears on every single page and every single post, or when page.php and single.php files are executed.
we do not recommend to show ads on all pages, because to many ads will appear on your page.

It also can be used to display images, banners or anything that contains HTML/Javascript.

== Installation ==

1. Unzip the installer and upload the resulting 'insert-ads-on-post' folder into the `/wp-content/plugins/` directory.  Alternatively, go to Admin > Plugins > Add New and enter 'Insert Ads On Post' in the search box.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Start inserting your ads.
